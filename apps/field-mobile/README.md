# App: Field Mobile (Operativa)

Herramienta diseñada para el personal de pista, cronometristas y veterinarios en la **gestión de competencias ecuestres**.

## Características
- **Toma de Tiempos:** Precisión de milisegundos en cada checkpoint.
- **Mesa Veterinaria:** Ingreso de datos clínicos y gestión de rechequeos.
- **Modo Offline:** Registro de datos sin conexión a internet y sincronización inteligente al recuperar señal.
