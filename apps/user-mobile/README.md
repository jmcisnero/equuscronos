# App: User Mobile (Espectadores)

Aplicación móvil desarrollada en **React Native + Expo** para el seguimiento en tiempo real de las competencias ecuestres.

## Propósito
Brindar transparencia y cercanía al público y a los competidores, permitiendo seguir cada segundo de la carrera desde cualquier lugar.

## Características para el Usuario
- **Live Tracking:** Resultados en vivo, promedios de velocidad y posiciones parciales.
- **Relojes de Carrera:** Cuenta regresiva personalizada para la próxima largada del jinete.
- **Notificaciones Push:** Alertas sobre cambios de estado, aprobaciones veterinarias y avisos de salida.
- **Historial Deportivo:** Consulta de desempeños anteriores de binomios y caballos.

## Visualización
Optimizado para mostrar datos técnicos de forma sencilla, facilitando la interpretación de pulsaciones, tiempos de recuperación y penalizaciones.
